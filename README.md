# Telecom-X---An-lisis-de-Evasi-n-de-Clientes
Proyecto de análisis de datos sobre churn de clientes en Telecom X. Se recopilan, limpian y analizan datos utilizando Python y bibliotecas como Pandas y NumPy para identificar factores que influyen en la cancelación de clientes y generar insights que apoyen el desarrollo de modelos predictivos y estrategias de retención.
